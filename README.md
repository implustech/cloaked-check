# cloaked test

This project was test for cloaked result.

## Getting Started

### Prerequisites

- [Git](https://git-scm.com/)
- [yarn](yarnjs.org) install as global
- [Node.js and npm](nodejs.org) Node ^4.2.3, npm ^2.14.7
- [phantomjs](phantomjs.org) version 2.1.1

### Developing

1. Run `yarn` to install server dependencies.

## Build & development

Run `yarn start` for start the ad server MVP.
